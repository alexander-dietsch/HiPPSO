Detailed Description

This class implements the bound handling strategy absorption for the particle swarm optimization.

If a particle would leave the search space, the new position is set to the intersection of the search space bound and the line from the old position to the new position.

void highprecisionpso::BoundHandlingAbsorption::SetParticleUpdate ( Particle *  p)
virtual

Updates the position of the particle p according to the current position and the new (already calculated) velocity.

The new position is calculated by adding the velocity to the position. If the new position would be outside of the search space bounds then the bound handling strategy absorption is activated. Afterwards the specified velocity adjustment is activated with suitable parameters.

Parameters
pThe particle, which should be updated.
